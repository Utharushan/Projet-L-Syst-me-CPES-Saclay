# Contexte 

En informatique théorique, un [L-système ou système de Lindenmayer](https://fr.wikipedia.org/wiki/L-Syst%C3%A8me) est un système de réécriture ou grammaire formelle, inventé en 1968 par le biologiste hongrois Aristid Lindenmayer.\
Un L-système modélise le processus de développement et de prolifération de plantes ou de bactéries.
